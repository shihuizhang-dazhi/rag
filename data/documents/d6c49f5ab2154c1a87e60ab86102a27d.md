# OWASP Top 10: A01 - 访问控制失效 (Broken Access Control)

## 概述

访问控制失效是 OWASP Top 10 2021 排名第一的安全风险。当用户能够执行超出其预期权限的操作时，就发生了访问控制失效。访问控制强制实施策略，使用户无法在其预期权限之外进行操作。

## 常见漏洞场景

### 1. 水平越权 (IDOR)
- 通过修改 URL 参数中的 ID 访问其他用户数据
- 例如：`/api/orders/123` 改成 `/api/orders/124` 即可查看他人订单

### 2. 垂直越权
- 普通用户执行管理员操作
- 例如：普通用户直接访问 `/admin/user/delete` 接口

### 3. 目录遍历
- 通过 `../` 访问服务器任意文件
- 例如：`/download?file=../../../etc/passwd`

### 4. CORS 配置不当
- 允许任意域跨域请求携带凭据
- 导致敏感数据被第三方网站窃取

### 5. JWT 令牌未校验
- 不验证签名或使用弱密钥
- 攻击者可伪造任意用户身份

## 检测方法

```bash
# IDOR 测试
curl -H "Authorization: Bearer <token>" https://target.com/api/user/1001

# 目录遍历
curl https://target.com/download?file=../../etc/passwd

# 未授权接口探测
ffuf -w common_paths.txt -u https://target.com/FUZZ
```

## 防御措施

- **默认拒绝**：除公共资源外，默认拒绝所有访问
- **服务端校验**：每次请求都在服务端验证权限，不依赖前端隐藏
- **最小权限原则**：用户仅获得完成任务所需的最小权限
- **记录与监控**：记录所有访问控制失败事件
- **JWT 安全**：使用强密钥、验证签名、设置合理过期时间
- **CORS 白名单**：仅允许受信任域跨域访问

## 实际案例

2023 年某电商平台因 IDOR 漏洞，攻击者遍历订单号获取了 50 万用户的姓名、地址和电话信息，被罚款 200 万元。
